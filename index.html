Board Room

A web app that helps student to locate Boarding Houses

The BoardRoom is a user-friendly platform that helps students find accommodation near their universities. The students can use BoardRoom to browse and select a wide range of houses that are tailored to their preferences and budget. BoardRoom provides students with detailed information, photos, and user reviews to help students find their perfect accommodations during their school year. BoardRoom aims to simplify the way students find their temporary accommodation during their school year.

Cosmos Cafe

An E-Commerce website that sells Coffee

In my portfolio project, 'Cosmos Cafe,' I've harnessed the capabilities of Next.js, Node.js, and PostgreSQL to create a dynamic and secure eCommerce platform. This project showcases my proficiency in full-stack web development, with Next.js providing a lightning-fast and intuitive user experience. Node.js is at the heart of the backend, facilitating seamless transactions and order processing. I've also implemented PostgreSQL to manage and store data, ensuring a robust and scalable database structure. 'Cosmos Cafe' is a testament to my ability to build a sophisticated eCommerce solution that caters to coffee enthusiasts while utilizing the latest web technologies for a seamless and secure user experience.

IWatchFilms

An Movie Review Website

In my portfolio project 'IWatchFilms,' I've leveraged the power of Next.js and Strapi to create a seamless movie review platform. This project showcases my skills in web development, with a strong emphasis on authentication and security. The frontend is built using Next.js, providing a fast and responsive user interface, while Strapi, an open-source headless CMS, ensures secure and user-friendly authentication. Additionally, I've integrated MovieDatabase APIs to fetch and display movie information, enhancing the user experience. 'IWatchFilms' exemplifies my ability to combine these technologies to build a robust and engaging web application for movie enthusiasts.

  • Category: Article Based Website

Student Manager

Student Management System

In my portfolio, I'm excited to present the 'Student Manager System,' a project that exemplifies my expertise in distributed systems and database integration. This system is built using Java RMI (Remote Method Invocation) to enable seamless communication between different parts of the application, ensuring a responsive and efficient student management solution. MySQL, a robust relational database management system, is employed to store and manage student data, providing a secure and organized repository for information. The 'Student Manager System' showcases my ability to design and develop complex, yet user-friendly, systems that streamline student information management and highlight my proficiency in Java RMI and database integration
